Detailed Description

The Window class is used to display and configure each element of Thunderpad, such as the Editor widget.

The Window is in charge of connecting and configuring everything together, such as the actions of the menubar with the functions of the text editor. Finally, the Window allows the Application class to create a new window and open a file using the configureWindow() function.
