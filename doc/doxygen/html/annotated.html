Data Structures
Here are the data structures with brief descriptions:
 CAboutThe About class implements a QDialog that shows information about the current version of Thunderpad
 CApplicationThe Application class is in charge of loading and configuring a new Window when Thunderpad is launched or when the system requests Thunderpad to open a file (for example, when the user selects "Thunderpad" from the Open with..
 CEditorThe Editor class implements a simple code editor widget that can be customized using XML files
 CLineNumberAreaImplements a simple QWidget that the Editor uses to display line numbers
 CMenuBarThe MenuBar class is in charge of creating and configuring a new QMenuBar widget to be used with the Window class
 CStatusBarThe StatusBar class is in charge of creating and configuring a new QStatusBar widget to be used with the Window class
 CSyntaxHighlighterThe SyntaxHighlighter is in charge of highlighting the contents of the document of the Editor and highlighting relevant matches, such as keywords, types and comments
 CThemeThe Theme class is in charge of reading the specified colors of an inputed XML theme and returning the individual values of each color used to theme the Editor widget
 CToolBarThe ToolBar class is in charge of creating and configuring a new QToolbarar widget to be used with the Window class
 CUpdaterThis class is in charge of checking for new updates and (if necessary) notify the user about a new version of WinT Messenger
 CWindowThe Window class is used to display and configure each element of Thunderpad, such as the Editor widget
